# React app run

In here we will show and explain the android app running

First, open a browser and type this URL: "<http://localhost:3000>"

## Unsigned User Page

This page is the first page you see when you open the app

![image](https://github.com/user-attachments/assets/b88434d5-2784-40eb-b373-89c898cd1aaa)

In this page you have an option to sign in, or to register, by clicking on the Sign up button

## Register Page

In this page the user can register himself, or go to the login page

![image](https://github.com/user-attachments/assets/8781f2d4-6041-45c1-b215-a515d173713f)

To register you must enter all the fields:

1. username
2. user display name
3. password at least 8 chars
4. confirm password
5. user profile image browse button to choose the image

## Login Page

In this page the user can login himself, or go to the register page

![image](https://github.com/user-attachments/assets/5f189fe5-be4f-4888-ad0e-d07f94ae3a7e)

For login you must enter the correct username and password to login

## Main Page

In this page you see some movies organized by categories that got from the api server, and a random movie is played.

![image](https://github.com/user-attachments/assets/da28d8b2-7592-45ae-b929-4967e81ceba7)

![image](https://github.com/user-attachments/assets/6e17d9e4-6395-44bf-b3cd-600175fb61dd)

In this page you have the toolbar where you have buttons that can:
logout, change dark\light mode, search movie, go to admin panel if user is admin.

## Admin Page

To get to this page you must be an admin (go to api docs to see more how to be an admin),
in this page you have the option to create/edit/delete movies and categories

![image](https://github.com/user-attachments/assets/c5f45c20-05cf-4a2f-82f2-2cead262e26d)

![image](https://github.com/user-attachments/assets/1ac2b8da-8b94-42a7-95e9-de37be9b0053)

In this page you have the toolbar where you have buttons that can:
logout, change dark\light mode, search movie/category, go back to home page.

## Movie details page

![image](https://github.com/user-attachments/assets/6586e41a-020e-4074-969d-c1f5380b3b15)

In this page, you can see the description of the movie and a button to watch the movie.
In addition, you can see the recommended movies for the user. The toolbar is identical to the one in the home page.

## Movie watch page

In this page you see the nane of the movie on top, the video player of the movie, and the description under the player.
After watching a movie in this page, the movie will be added to the Watched Movies array of the user.
The toolbar is identical to the one in the home page.

![image](https://github.com/user-attachments/assets/4197bf88-592b-4eee-bd59-f2cc74a2c21c)
